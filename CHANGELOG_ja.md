# 変更履歴

このプロジェクトへの注目すべき変更点はこのファイルに記録されます。

この形式は [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) に基づいており、
このプロジェクトは [セマンティック バージョニング](https://semver.org/spec/v2.0.0.html) に準拠しています。

## [0.1.0] - 2026-01-25

### 追加
- 自動化されたバージョンアップワークフローとプロジェクトルールの追加。
- Mermaid ダイアグラムを含むソフトウェアアーキテクチャドキュメントの作成。
- ビルドの最適化と開発者体験向上のための pnpm および Turbo の導入。
- クリーンアーキテクチャ実現のための `IdGenerator` インターフェースと `CryptoIdGenerator` 実装。
- ズームリセット機能とショートカットキー設定の強化。
- ボーダーおよび接続線のカスタムスタイル継承機能。
- スタイル関連 API への詳細な TSDoc コメントの追加。

### 変更
- `domain`, `application`, `presentation`, `infrastructure` の各レイヤーへのディレクトリ構造の再編。
- パッケージ管理ツールを `npm` から `pnpm` へ移行。
- `Kakidash` コンストラクタを拡張し、`maxNodeWidth` や `customStyles` をオプションで受け取れるように変更。

### 修正
- 重複したショートカット処理の削除による、貼り付けが二重に行われる問題の修正。
- ズームリセット時の表示中心の計算ロジックの修正。
- VSCode 埋め込み時のダークモードにおけるノード背景色の表示改善。
- `maxWidth` 適用時にテキストが縦長に表示される問題の修正。
- 葉ノードにおける誤った折りたたみを防ぐためのロジックの微調整。
